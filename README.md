# 🎂 Fig & Flour — Luxury Artisan Bakery Website

> *"Every Dessert Tells A Story"*

A premium, award-winning bakery website concept built for **Fig & Flour**, a fictional luxury Parisian-style patisserie based in Mumbai. Designed to impress — inspired by Awwwards, Apple, and top French dessert brands.

---

## ✨ Live Preview

Open `index.html` in any modern browser — no build step required.

---

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Structure | Semantic HTML5 |
| Styling | Vanilla CSS (2,676 lines) — zero frameworks |
| Logic | Vanilla JavaScript (ES6+) |
| Fonts | Fraunces · Playfair Display · Jost · Inter |
| Images | AI-generated editorial photography |

---

## 🎨 Design Features

- **Cinematic hero** with parallax background & floating pastry animations
- **Custom magnetic cursor** with gold ring tracker
- **Loading screen** with animated progress bar & floating crumbs
- **Glassmorphism navbar** — transparent → frosted glass on scroll
- **Dark mode toggle** with smooth theme transition
- **Scroll progress indicator** with gradient bar
- **Mouse glow effect** following the cursor
- **3D card tilt** on product cards
- **Text reveal animations** staggered on page load
- **Smooth scroll** throughout all sections

---

## 🛒 Full E-Commerce System

- 12 products across 8 categories (Cakes, Pastries, Macarons, Cheesecakes, Cookies, Brownies, Tarts, Coffee)
- **Slide-in cart drawer** with live totals, GST (5%), delivery fee
- **Coupon codes**: `SWEET10` · `WELCOME` · `FIRSTORDER` · `FIGFLOUR`
- **Gift wrap** option (+₹50)
- **Checkout form** — Name / Phone / Address / Payment Method (Cash · UPI · Card)
- **Order confirmation** with animated confetti 🎉 and unique order number
